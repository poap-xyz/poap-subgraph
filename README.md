# POAP Subgraph

This Subgraph sources events from the POAP contract in different networks.

## Deploying the subgraph:

**First time only**
```ssh
yarn install
```

Available networks: mainnet, xdai, sokol, kovan, ropsten 

```ssh
yarn prepare:<network>
yarn codegen
yarn build
yarn deploy:<network>
```
